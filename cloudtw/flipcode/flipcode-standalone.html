FlipCode 1.0.0
磚塊式程式編輯器
FlipCode適合正式開始學習學程邏輯的磚塊式程式編輯工具，以Arduino為基礎，易用卻嚴謹的C語言架構，可以直接燒錄至機器人來執行相對應的動作，並且可以進行各式有趣的專題學習。

支持硬體：OSX／Win7及以上／iOS 9以上作業系統。
配合學具：E300、E310

  • 線上版
  • Windows App
  • Mac App
  • iOS App
1. 環境設置-安裝Extension
線上版搭配Chrome瀏覽器使用，因此在開始使用各項線上學習工具及韌體還原之前，請依照您的作業系統下載相對應的Extension(擴充套件)及Driver(趨動程式)，即可輕鬆在線上開始使用各項學習工具。

2. 測試環境-燒錄測試韌體
當你完成了Extension(擴充套件)及Driver(趨動程式)的安裝，請準備好你的機器人學具，並使用USB連結你的機器人學具與電腦，選擇板子及連接埠，並按下燒錄。

1. Windows 單機版 App 說明
- FlipCode Windows 單機版 App 提供你在無網路環境下，也可以執行機器人程式的編寫與寫入。更能符合學校場域的使用情況，不再受限網路連線問題。
- 適用作業系統：Windows 7, Windows 8, Windows 8.1, Windows 10
2. 操作流程指引
- 請點選「程式下載」下載 Windows 單機版 App 檔案，檔案名稱：FlipCode standalone App(windows)_ver1.0.0.zip
- 請依操作流程一步步完成下載。
- 安裝過程將依個人電腦安全性設定出現相關提示訊息，請選點「繼續執行」
- 完成單機版本安裝後，程式將自動偵測是否需加裝 USB Driver。
- 所有程式安裝完畢，即可開始啟動單機版App。

2. 操作流程指引
- 請點選「程式下載」下載mac單機版App檔案，檔案名稱：FlipCode standalone App(mac)_ver1.0.0.zip
- 請用右鍵點擊安裝檔，並點按"開啟"選項，即可開始安裝程序。 - 請依操作流程一步步完成下載。

- 所有程式安裝完畢，即可開始啟動單機版App。
