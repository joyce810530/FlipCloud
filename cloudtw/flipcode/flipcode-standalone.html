1. 安裝擴充套件(Extension)
在開始使用FlipCloud的各項服務（例如：FlipRAS和FlipCode）及韌體還原前，請先依照個人使用的作業系統，安裝擴充套件，依指示進行下載即可開始啟用各項功能。
  • Windows
  • MacOS
  • Chrome OS
  • Linux

Windows 7

EXTENSION
1. 下載 "擴充套件"。
2. 開啟chrome​ ​->​ ​點擊右上角的 ‘More’   ->​ 點擊’更多工具’​ ​->​ 並點擊‘擴充功能’。
3. 於‘擴充功能’視窗中拖曳並安裝第1步驟所下載的擴充套件。

DRIVER
1. 下載"驅動套件"。
2. 雙擊"驅動套件"並安裝。

2. 更新項目:

3. 任務完成
您已成功完成 FlipCode Windows版本應用程式的安裝。
如有任何使用上的問題，您可點選下方知識庫找到解答。
 
 
1. 關於 FlipCode 應用程式
支援作業系統：Mac10.10以上版本作業系統(64位元)
1. 版本：1.4.1(更新日期：2019年9月17日) 2. 更新項目: 　　1. 與FlipAIOT軟體串接更新
2. 安裝指引
1. 請點選「下載」按鈕下載FlipCode應用程式。
2. 請將步驟一下載的檔案拖曳至Mac系統的「應用程式」即完成安裝。
3. FlipCode應用程式將自動偵測您是否需另外安裝USB趨動程式，並提供您相對應的安裝指引。
4. 您也許需要至「安全性及隱私權」設定，允許安裝使用USB趨動程式。
3. 任務完成
您已成功完成FlipCode Mac版本應用程式的安裝。
如有任何使用上的問題，您可點選下方知識庫找到解答。
1. 關於FlipCode iOS應用程式
支援作業系統：iOS 9.3.5以上版本作業系統。
搭配硬體： iPad第3代，iPad第4代，iPad mini，iPad mini 2，iPad mini 3，iPad Air和iPad Air 2。
2. 安裝指引
1. 請至App Store搜尋 FlipCode。

2. 另可點選以下連結或掃描QR Code直接安裝FlipCode iOS應用程式。

3. 請點選「取得」即可安裝。
備註：此應用程式僅支援iPad上使用。
3. 藍牙配對說明
1. FlipCode iOS應用程式僅可搭配使用E310並具備藍牙裝置之機器人套件。在開始使用之前，請務必打開iPad的藍牙裝置，並開啟您的機器人。
2. 開啟FlipCode iOS應用程式並點選左上方藍牙「未連結」按鈕。
3. 在「配對裝置」的視窗中，請在可配對裝置的清單中，按下「連結」即可完成配對。
4. 任務完成
您已成功配對FlipRobot與iPad，請開始你的第一個FlipCode的程式，並利用iPad將程式順利燒錄至您的機器人吧！
如有任何使用上的問題，您可點選下方知識庫找到解答。
