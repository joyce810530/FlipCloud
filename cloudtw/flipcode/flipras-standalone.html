1. 安裝擴充套件(Extension)
在開始使用FlipCloud的各項服務（例如：ActionScript和FflipCode）之前，包括韌體還原，請先依照個人使用的作業系統，安裝擴充套件，依指示進行下載即可開始啟用各項功能。
  • Windows
  • MacOS
  • Chrome OS
  • Linux

Windows 7

EXTENSION
1. 下載 "擴充套件"。
2. 開啟chrome​ ​->​ ​點擊右上角的 ‘More’   ->​ 點擊’更多工具’​ ​->​ 並點擊‘擴充功能’。
3. 於‘擴充功能’視窗中拖曳並安裝第1步驟所下載的擴充套件。

DRIVER
1. 下載"驅動套件"。
2. 雙擊"驅動套件"並安裝。

3. 於‘擴充功能’視窗中拖曳並安裝第1步驟所下載的擴充套件。
