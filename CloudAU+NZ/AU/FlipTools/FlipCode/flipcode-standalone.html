FlipCode 1.0.0
Blockly Programming Software
FlipCode is a block-based programming learning tool to inspire you to improve your computational thinking skill. It is an Arduino-Based C language which you can use it to code and upload your program to your FlipRobot, and develop various of interesting projects with FlipCode and E300.

Support OS: OSX, Win7, 8, 8.1 and 10
Support Hardware : FlipRobot：E300、E310

  • Online Version
  • Windows App
  • Mac App
  • iOS App

2. Default firmware download testing
While you installed the Extension and driver, please get ready for your FlipRobot, USB connector and your laptop/PC. Please select your FlipBrain and port, and click upload.



3. Way to go!
Congregation! You totally know how to use the FlipCode online version right now. Please click the start button to start your new adventure of FlipCode, or you could click our Knowledge Base to get more information about FlipCode.
1. About FlipCode Windows standalone app
FlipCode Windows standalone app provides you can use FlipCode without internet. You can edit and upload your coding to FlipRobot. It’s more suitable for school solution.
Support OS: windows 7, 8, 8.1 and 10.
2. How to install
- Please click the download button to download FlipCode windows standalone app.
- Please follow the steps to install the app.
- Since the set up of each laptop or PC is quiet different, it may pop-up a window to check your security, please click the continue button to finish the install process.
- After the installation of FlipCode windows standalone app, we will automatically detect if you need to install the USB Driver.
- You could start your FlipCode windows standalone app now.
