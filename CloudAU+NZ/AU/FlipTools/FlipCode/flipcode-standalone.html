FlipCode 1.0.0
Blockly Programming Software
FlipCode is a block-based programming learning tool to help you to improve your programming and computational thinking skills. You can code in it and upload your program directly into your FlipRobot to complete various fun robotic activities and challenges.

Support operating systems: macOS (High Sierra 10.9.5 and above), Windows 7, 8, 8.1 and 10, Chrome OS, Linux Support hardware: FlipRobot E300/E310 starter kits and all current available extension kits

  • Online Version
  • Windows App
  • Mac App
  • iOS App
1. Install Extension
Since FlipCode Online Version runs only on Chrome browser, depending on the operating system you are in, you need to firstly install the FlipRobot extension into your Chrome browser as instructed below. It helps your Chrome browser to find the corresponding FlipRobot USB port once it’s connected with your computer.

2. Default firmware download test
Once you complete the above installation procedures, let’s test it to see if you have install the FlipRobot extension into your Chrome browser successfully. Please switch on the FlipRobot and connect it with your computer using the micro USB cable provided. Then select the relevent E300/E310 board and FlipRobot (FR) port as shown below, then click on Upload to see if you are able to successfully upload the test default firmware into the FlipRobot.

1. About FlipCode Windows app
Once you install the FlipRobot Windows app into your computer, you can then create programs there and upload them into your FlipRobot directly with no internet connection required.
Support OS: windows 7, 8, 8.1 and 10.
2. How to install
- Please click the Download button below to download the FlipCode Windows app.
- Please then follow the prompted steps to install the app accordingly.
- If a security warning/reminder window pops up, please click the Continue/OK button to complete the installation process.
- After the installation, FlipCode software will automatically detect if you need to install the USB Driver and prompt you with required actions accordingly.

Once you install the FlipRobot Mac app into your computer, you can then create programs there and upload them into your FlipRobot directly with no internet connection required.
Support OS: macOS High Sierra 10.9.5 and above.
2. How to install
- Please click the Download button to download the FlipCode mac app.
- Please right click the dmg file and select "open" to start the install process.
- Please then follow the prompted steps to install the app accordingly.
- After the installation, FlipCode software will automatically detect if you need to install the USB Driver and prompt you with required actions accordingly.
- Depending on your computer settings, you may need to go to your compute’s privacy and security settings to allow the USB driver installation.
