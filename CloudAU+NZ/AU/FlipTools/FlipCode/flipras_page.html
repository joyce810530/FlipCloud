2. Default firmware download test
Switch on the FlipRobot and connect it with your computer using the micro USB cable provided. Select the board (E300) and port (FR Device) , then click on Upload.

You have now successfully installed the FlipRobot extension into your Chrome browser.

1. About FlipRAS Windows app
Supporting OS: Windows 7, 8, 8.1 and 10.
2. How to install
1. Click the Download button below to download the FlipRAS Windows app.
2. If a security warning/reminder window pops up, click the Continue/OK button.
3. FlipRAS software will then automatically detect if you need to install the USB Driver and prompt you with required actions.
3. Mission accomplished
You have now successfully installed the FlipRobot extension into your Chrome browser.
If you still have any questions, click on the Knowledge Base icon below.
 
 
1. About FlipRAS Mac app
Supporting OS: macOS High Sierra 10.9.5 and above.
2. How to install
1. Click the Download button to download the FlipRAS mac app.
2. Right click the dmg file and select “open” to start the installation process.
3. FlipRAS software will automatically detect if you need to install the USB Driver and prompt you with required actions.
4. You may need to go to privacy and security settings to allow the USB driver installation.
3. Mission accomplished
You have now successfully installed the FlipRobot Mac app.
If you still have any questions, click on the Knowledge Base icon below.
